# code-child

코린이를 위한 학습자료 가이드 사이트로, [박윤종](https://github.com/yoonjong-park) 포트폴리오를 겸하여 제작되었음.

## 접속 URL

https://codechild.vercel.app/

## 제작 동기

1. portfolio 사이트 제작
2. 내가 작성한 글과, 좋았던 글을 리스트 형태로 바로바로 보고 싶어서 정리

## 향후 과제

1. 검색기능 추가 - 자료의 양이 많아질 경우
2. 페이지 추가 - 좋았던 영상도 정리

## 사용한 라이브러리

1. Next.js 13
2. React.js

## 배포

Vercel을 이용하여 배포함.
